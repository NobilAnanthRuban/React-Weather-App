# Neo Weather Dashboard 2025

Futuristic neon glassmorphism weather and air quality dashboard built with React, Tailwind, Open‑Meteo, OpenAQ, and Geoapify.

## Features

- Dark / light theme with neon accents and smooth transitions (stored in `localStorage`).
- Geolocation-based current weather (temperature, condition, humidity, wind, visibility) via Open‑Meteo.[web:10][web:14]
- Air quality index, PM2.5, PM10, NO₂, O₃ via OpenAQ nearby stations.[web:15][web:19]
- City search with autocomplete suggestions from Geoapify Geocoding.[web:12]
- Dashboard layout with sidebar, top search bar, main weather card, AQI card, hourly timeline slider, and 7‑day forecast.
- Multi-page navigation with React Router: Dashboard, Forecast, Air Quality, Settings.
- Mobile-first responsive design with glassmorphism and hover animations.
